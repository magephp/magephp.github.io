Parses YAML into a PHP array.

Parses YAML into a PHP array.

The parse method, when supplied with a YAML stream (string or file), will do its best to convert YAML in a file into a PHP array. Usage:

$array = Yaml::parse('config.yml');
  print_r($array);

As this method accepts both plain strings and file names as an input, you must validate the input before calling this method. Passing a file as an input is a deprecated feature and will be removed in 3.0.

Parameters

$input
Path to a YAML file or a string containing YAML
$exceptionOnInvalidType
True if an exception must be thrown on invalid types false otherwise
$objectSupport
True if object support is enabled, false otherwise
