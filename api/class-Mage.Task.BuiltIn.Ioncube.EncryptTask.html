Class EncryptTask

This allows intergrating IonCube PHP encoder into deployment system It takes the source path renames it to .raw creates a fresh source path and runs ioncube encoder placing encoded files into source folder. Afterwards it removes the old .raw folder This means that we dont have to change the source path within the main scripts and allows the built in rsync and other tasks to operate on the encrypted files.

IonCube PHP Encoder can be downloaded from http://www.actweb.info/ioncube.html
